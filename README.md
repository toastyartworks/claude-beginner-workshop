# Claude Code 스타터 워크스페이스

Claude Code 워크샵 참가자를 위한 시작 템플릿입니다.

---

## 시작하기

```bash
git clone https://github.com/toastyartworks/claude-beginner-workshop.git ~/my-workspace
cd ~/my-workspace
claude
```

---

## 폴더 구조

```
00-inbox/       빠른 메모, 임시 저장
10-work/        진행 중인 업무
20-resources/   참고 자료
30-personal/    개인 메모
90-archive/     완료된 업무
```

---

## 첫 번째 할 일

Claude Code를 실행하고 이렇게 말해보세요:

```
나는 코딩을 모르는 비개발자야.
지금부터 CLAUDE.md 파일을 작성하고 싶어.
내 역할, 업무, 선호 스타일을 파악할 수 있도록 질문을 하나씩 던져줘.
```

`CLAUDE.md`는 AI에게 "나는 누구고, 어떤 일을 하는지" 알려주는 설정 파일입니다.
한 번 만들어두면 매번 설명하지 않아도 됩니다.
