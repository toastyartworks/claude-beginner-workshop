`00-inbox/` 폴더를 정리해주세요.

**수행할 작업:**

1. `00-inbox/` 폴더를 Glob으로 스캔해서 파일 목록 확인
2. **비어있으면**: "inbox 비어있어! 정리할 게 없어." 안내 후 종료
3. 각 파일 내용을 Read로 읽어서 주제·성격 파악
4. 파일별로 이동할 폴더 제안:
   - `10-work/` — 진행 중인 업무 관련
   - `20-resources/` — 참고 자료, 레퍼런스
   - `30-personal/` — 개인 메모, 일기, 생각 정리
   - `90-archive/` — 이미 완료되었거나 더 이상 필요 없는 것

5. 제안 형식:
   ```
   ## inbox 정리 제안
   | 파일 | 내용 요약 | 이동 제안 |
   |-----|----------|----------|
   | 메모.md | 프로젝트 A 관련 아이디어 | → 10-work/ |
   ```

6. **반드시 사용자 확인을 받은 후** `mv` 명령으로 이동
7. 애매한 파일은 "이 파일은 어디로 보낼까?" 하고 물어보기
8. 이동 완료 후 결과 요약 출력

**핵심 규칙:**
- 자동 이동 절대 금지 — 반드시 사용자 확인 후 이동
- 확인 없이 파일을 옮기거나 삭제하지 않기
- .DS_Store 등 시스템 파일은 무시
