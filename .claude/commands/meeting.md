미팅노트를 만들어주세요.

**인자**: $ARGUMENTS
**오늘 날짜**: $CURRENT_DATE

**수행할 작업:**

1. 미팅 제목 결정:
   - **인자가 있으면**: 인자를 미팅 제목으로 사용
   - **인자가 없으면**: "어떤 미팅이야?" 물어보기

2. `10-work/meetings/` 폴더가 없으면 생성
3. `10-work/meetings/$CURRENT_DATE-[제목].md` 파일 생성

**템플릿:**

```markdown
# [미팅 제목]
**날짜**: $CURRENT_DATE
**참석자**:

## 안건
-

## 논의 내용
-

## 결정사항
-

## 액션아이템
- [ ]
```

4. 생성 후 물어보기: "안건이나 참석자 미리 적어줄까?"
5. 사용자가 자유롭게 메모를 던지면 → 적절한 섹션에 구조화해서 정리

**핵심 규칙:**
- 사용자가 비구조적으로 말해도 알맞은 섹션에 구조화해서 정리
- 액션아이템은 반드시 체크리스트(`- [ ]`) 형태
- 미팅 끝나면 "액션아이템 정리해줄까?" 제안
