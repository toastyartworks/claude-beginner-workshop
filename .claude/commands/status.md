`10-work/` 폴더의 프로젝트 현황을 대시보드로 보여주세요.

**수행할 작업:**

1. `10-work/` 하위 폴더·파일을 Glob으로 스캔
2. **비어있으면**: "10-work/에 프로젝트가 없어. 새 프로젝트를 시작해볼까?" 안내 후 종료
3. 각 프로젝트 폴더의 내용을 Read로 읽어서 상태 파악:
   - 파일 수정일 기준으로 최근 활동 판단
   - 내용에서 상태 키워드 확인 (진행중, 완료, 대기 등)
4. 대시보드 형태로 출력

**출력 형식:**

```
## 프로젝트 현황

| 프로젝트 | 상태 | 최근 파일 | 한 줄 요약 |
|---------|------|----------|-----------|
| 프로젝트A | 진행중 | 2026-03-20 | 기획안 작성 중 |
| 프로젝트B | 대기 | 2026-03-15 | 클라이언트 피드백 대기 |
```

**상태 판단 기준:**
- 최근 3일 내 수정 → 진행중
- 3~14일 → 대기
- 14일 이상 → 멈춤
- 내용에 "완료", "done" 등 키워드 → 완료

**핵심 규칙:**
- 읽기 전용 — 아무것도 수정하지 않음
- 파일 생성·수정·삭제 절대 금지
