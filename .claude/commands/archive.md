완료된 작업을 아카이브해주세요.

**인자**: $ARGUMENTS

**오늘 날짜**: $CURRENT_DATE

**수행할 작업:**

1. **인자가 있으면**: 해당 프로젝트/파일을 아카이브 대상으로 지정
2. **인자가 없으면**: `10-work/` 스캔 → 완료된 것 같은 항목을 찾아서 제안
   - 오래된 파일, "완료"/"done" 키워드 포함 등

3. 아카이브 대상 확인:
   - 파일/폴더 내용을 읽어서 한 줄 요약 생성
   - 이동 계획 안내:
     ```
     아카이브 대상: 프로젝트A
     요약: 브랜딩 프로젝트 - 최종 전달 완료
     이동: 10-work/프로젝트A → 90-archive/2026-03-20-프로젝트A
     ```

4. **반드시 사용자 확인 후** 이동 실행:
   - `90-archive/` 폴더가 없으면 생성
   - 파일명 앞에 완료 날짜 추가: `$CURRENT_DATE-원래이름`
   - `mv` 명령으로 이동

5. 이동 후 결과 안내: "아카이브 완료: 90-archive/$CURRENT_DATE-프로젝트명"

**핵심 규칙:**
- 반드시 사용자 확인 후 이동
- 확인 없이 파일을 옮기거나 삭제하지 않기
- 이동 전 내용 요약을 반드시 보여주기
